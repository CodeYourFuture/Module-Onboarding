What is a purpose of a Wireframe?

A Wireframe is a simple two dimensional outline of a Webpage. It is used mainly by Web Developers to communicate and show users how a website will look like. It is just a visual frame of a website

Learn more about Wireframes

What is a purpose of a Readme File?

A Readme File is a brief description or instruction to a Website or an App in text file format. Its purpose is for a User to know what are the contents of a Website or an App.
