Purpose of a README File

A README File is a crucial piece of documentation that explains what the project is, what it does and why it exists. It serves as the main introduction and guide to a project, providing instructions and making the project easily accessible. Its essence is to help users understand, utilize and potentially contribute to a project.

Explaining a Branch in Git

In Git, a branch is a snapshot of your project at a certain point in time. It is essentially a lightweight movable pointer to a commit. When you create a new branch, git creates a pointer that can move independently on that branch. As you make commits on that branch, the pointer advances forward with each commit. pointer advances forward with each commit.
