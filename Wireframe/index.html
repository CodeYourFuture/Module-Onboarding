What is README.md file?

A README.md file is a text file written in Markdown (.md) that provides essential information about a project—especially common in software development and open-source repositories (like on GitHub).

What It's For:

  • the project (what it is and why it exists)
  • Explain how to install, use, or contribute
  • Document dependencies or requirements
  • Provide contact or licensing information

What is a branch in GitHub?

In GitHub (and Git in general), a branch is a separate line of development that lets you work on changes without affecting the main (usually main or master) codebase.

Github branch

What is wireframes?

A wireframe is a visual guide or blueprint that outlines the structure of a web page, app, or user interface—without focusing on colors, graphics, or final design elements.
