Web Fundamental

In this article, we’ll explain the purpose of some essential terms used in web development and Git.

README file

A README file is a key component of a GitHub repository. It usually provides an overview of the project, installation steps, usage instructions, and other important details to help users understand and use the project effectively

wireframe

A wireframe is a visual guide or blueprint that represents the basic structure and layout of a webpage or application interface. It focuses on what goes where.
