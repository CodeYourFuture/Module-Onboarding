What is the purpose of a README file?

A README file is a document that provides essential information about a project, such as what it does, how to install and use it, and any dependencies or configurations needed.
It serves as the first point of reference for anyone looking at the code and is crucial for open-source projects.

What is the purpose of a wireframe?

A wireframe is a visual guide or blueprint that outlines the basic structure and layout of a webpage or app.
It helps designers and developers plan the content, navigation, and functionality before diving into design or development.

What is a branch in Git?

A branch in Git is a separate line of development that allows you to work on new features or bug fixes without affecting the main project.
Once the changes are ready, branches can be merged back into the main codebase, keeping the project organized and flexible.
