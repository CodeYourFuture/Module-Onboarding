What is the purpose of a README file?

A README file serves as an introductory document for a project,providing essential information to users and contributors about the project, its purpose, how to use it, and how to get help.it's typically the first file a user will encounter when exploring a repository or software project ,offering a concise overview and instruction.

what is a branch in Git?

A branch in Git is a lightweight, movable pointer to a specific commit in your project history. It allows you to work on different features or fixes separately from the main codebase.
