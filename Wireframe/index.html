What is the purpose of a README file?

A README file is a simple guide that explains what a project is,
how to set it up, how to use it, and other helpful details for anyone working with it.
It is usually the first file a user sees when they open a project,
and acts as a guide for anyone who wants to understand or contribute to the project.
