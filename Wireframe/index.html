What is the purpose of a wireframe?

A wireframe is a visual blueprint used in web design to outline the basic structure and layout of a webpage or app interface before any visual design or content is added. It help designers, developers, and stakeholders communicate ideas clearly, align expectations, and identify usability issues early in the design process.

What is a Branch in Git?

A branch in Git is a lightweight, movable pointer to a specific commit, allowing developers to work on different features, fixes, or experiments independently of the main codebase. It enables multiple versions of a project to exist simultaneously without interfering with each other.
