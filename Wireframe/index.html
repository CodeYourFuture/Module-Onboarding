  1. Explaining what the project is about
  2. Giving instruction on how to install,use,or contribute to the project
  3. Listing prerequisites,technology used,and authors

WireFrame

A WireFrame is a visual guide like a blueprint used in web or app design to:

  1. Show the layout and structure of a page or screen
  2. Plan where content,images,buttons,and features will go
  3. Help developers agree on the structure before real design and coding begin

Branch in Git

A Branch in Git is a separate line of Development.It allows you to:

  1. Work on new features or fixes without affecting the main codebase
  2. Experiment freely and merge changes later
  3. Keep the main branch stable
