What is the purpose of a README file?

A README serves as a guide that gives users a detailed description of a project and includes guidelines on how to install and run a project.

What is the purpose of a wireframe?

A wireframe is a rough schematic created in the early stages of digital product design to help visualize and communicate the structure of a product or website.

What is a branch in Git?

A branch is a sequence of commits in a project, or in other words a pointer to a snapshot of commits to a project.
