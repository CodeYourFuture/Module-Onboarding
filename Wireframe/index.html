What is the purpose of a README file?

A README file is a file you can add to your GitHub repository (repo) to inform people interested in your project about its usefulness, how it can be used and what they can do with it. According to GitHub “You can add a README file to a repository to communicate important information about your project. A README, along with a repository license, citation file, contribution guidelines, and a code of conduct, communicates expectations for your project and helps you manage contributions.”
