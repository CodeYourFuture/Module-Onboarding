README File

A README file serves as a guide for a project, providing essential information such as its purpose, usage instructions, installation steps, and contributors. It helps others understand and engage with the project effectively.

Wireframe

A wireframe is a visual blueprint for a website or application, outlining its structure, layout, and key elements. It helps designers and developers plan and communicate ideas before actual development begins.

Branch in Git

A branch in Git is a separate line of development, allowing multiple features, bug fixes, or experiments to be worked on independently. This enables developers to collaborate and merge changes without impacting the main codebase.
