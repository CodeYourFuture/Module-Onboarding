What is the purpose of a README file?

A README file explains the purpose , features,and usage of a project. It's often the first thing users or developers read to understand how to use or contribute to the project.

What is the purpose of wireframe?

A wireframe is a basic visual guide used in UI/UX design to outline the structure and layout of a webpage or app without focusing on design details like colors or images.

What is a branch in Git ?

A branch in Git allows developers to work on different features or fixes independently without affecting the main codebase, enabling safe collaboraton and
experimentation.
