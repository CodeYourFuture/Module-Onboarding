What is a README File?

A README file is a document that provides essential information about a software project. It is meant to explain the project's purpose, installation instructions, usage guidelines, and how others can contribute. It is usually written in plain text and serves as a guide for developers or users to understand and work with the project.

What is a Wireframe?

A wireframe is a basic visual guide that represents the structure and layout of a webpage or app, focusing on content placement and functionality rather than design details.

What is a Branch in Git?

A branch in Git allows you to work on different versions of a project. It provides a parallel workspace where you can develop new features or fix bugs without affecting the main project.
