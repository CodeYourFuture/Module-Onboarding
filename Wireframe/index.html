What is the purpose of a wireframe?

A wireframe is a basic visual representation of the layout and structure of a webpage or app. Its purpose is to outline the placement of elements like headers, navigation, content, and buttonswithout focusing on design details. Wireframes help:

  • Plan the structure and functionality of a project.
  • Communicate ideas clearly to stakeholders or team members.
  • Ensure the layout aligns with user needs before starting the actual design and coding.

What is a branch in Git?

A branch in Git is like a parallel line of development within a repository. It allows you to work on new features, bug fixes, or experiments without affecting the main codebase.Key points about branches:

  • Main Branch: Often represents the production-ready or stable code
  • Feature Branch: Created to isolate work on a specific feature or task.
  • Benefits: It helps developers collaborate without overwriting each other's work.
