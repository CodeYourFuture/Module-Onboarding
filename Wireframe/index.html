What is the purpose of a README file?

A README file provides an overview of a project, instructions on how to set it up, use it, and contribute to it.
It is the first point of reference for anyone working with the project.

What is the purpose of a wireframe?

A wireframe is a simple visual guide that outlines the structure and layout of a webpage or application. It helps teams plan design and functionality before development begins.

What is a branch in Git?

A Git branch is a parallel version of a project where developers can work independently without affecting the main code. Branches make it easier to develop features and fix bugs.
