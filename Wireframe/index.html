What is the purpose of a README file?

A README file provides an overview of a project, including its purpose, usage, and setup instructions. It's essential for helping others understand and contribute to your work.

What is the purpose of a wireframe?

A wireframe is a visual blueprint of a webpage or app. It helps designers and developers plan layout and functionality before coding begins.
