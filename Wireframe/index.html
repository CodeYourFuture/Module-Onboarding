What is the purpose of a README file?

A README file is added to a repository to communicate important information about your project. A README, along with a repository license, citation file, contribution guidelines, and a code of conduct, communicates expectations for your project and helps you manage contributions.

What is the purpose of a wireframe?

A wireframe is a visual guide to what a product should look like. Its main goal is to create an app or site that delivers a cohesive and well-designed experience. It is also a great way to ensure that every element in a UI has a purpose.

What is a Branch in Git?

In Git, a branch is a pointer to a specific commit, effectively creating a parallel development line within your repository. It allows developers to work on new features, fix bugs, or experiment with code changes without directly altering the main codebase.
